Aula 4 - Width

• Define a altura minima para o elemento.

• Width irá defini o tamanho da largura do elemento.

• Assim como para height, as propriedades de min e max width irá sobrepor a propriedade width.

• Caso max-height ou min-height estejam definidas, elas tem prioridade ao height.

• Valores possíveis: Auto | Length | Initial | Inherit | Fit-content | Max-content | Min-content | Revert | Unset.

• Valor padrão: auto.

• Valor herdado: não.
